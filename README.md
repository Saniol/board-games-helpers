# Board Games Helpers

Simple web helpers for board games.

## Public pages

- Landing page: https://saniol.github.io/board-games-helpers/
- Rumikub: https://saniol.github.io/board-games-helpers/rumikub.html
- Star Wars Rebel: https://saniol.github.io/board-games-helpers/star-wars-rebel.html

## rumikub

Open rumikub.html.

- Press C, D, J, K, M, N, S, T, U, or Z to speak the matching funny message.
- Press X for the penalty message.
- Press any other key to reset the timer.
- Click anywhere to reset the timer.

## star-wars-rebel

Open star-wars-rebel.html.

- No keyboard bindings are implemented.
- Use mouse clicks on the colored boxes or the Reset button.
